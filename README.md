Minesweeper made in python,
it uses random module to assign bombs to random location and then as per minesweeper rules we have to find out squares not containing bomb

controls:
1)left click to reveal
2)right click to place flag

the numbers on button represent the number of bombs the button is touching 0 meaning it touches 0 bombs 1 meaning it touches 1 and so on.
